## Setup Instructions

To run the Jupyter notebook and reproduce the analysis:

1. **Clone this repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. **Set up a virtual environment**
   ```bash
   python -m venv wordembedding_env
   source wordembedding_env/bin/activate  # macOS/Linux
   # wordembedding_env\Scripts\activate  # Windows
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Start Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

5. **Open the notebook**
   - Navigate to `wordembedding_analysis.ipynb` in your browser
   - Run all cells using "Cell" â†’ "Run All"
